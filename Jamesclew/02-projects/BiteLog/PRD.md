# BiteLog PRD — 디자인 리뉴얼 + 프로덕션 출시

> 작성일: 2026-04-06 | 버전: v1.0

## 1. 배경 및 문제 정의

### 왜 만드는가?
- **현재 문제점**: BiteLog는 22개 라우트, 31개 서비스를 갖춘 기능적으로 완성된 프로토타입이지만, 데모 데이터 의존·UI 미완성·테스트 부족으로 앱스토어 출시 불가 상태
- **시장 기회**: 한국 낚시 앱 시장은 물반고기반(100만+), 바다타임(50만+), 어신 등이 선점했으나, 대부분 예약·물때 특화로 **AI 기반 종합 낚시 도우미**는 부재. 해외 Fishbrain($75/yr, 2000만 유저)이 증명한 "AI 입질예보 + 커뮤니티" 모델의 한국판이 없음

### 해결하려는 핵심 문제 (1문장)
> 한국 낚시인이 출조 전·중·후에 필요한 모든 정보(날씨, 입질예보, 기록, 커뮤니티, 선단추적)를 AI 기반으로 한 앱에서 해결할 수 있게 한다.

### 우리의 차별화 포인트
1. **AI 트리플**: 입질예보(BiteTime) + 어종식별(Gemini Vision) + AI 컨시어지 — 한국 앱 중 유일
2. **Fleet Radar**: AIS 기반 실시간 선단 추적 — 안전 + 조황 포인트 추론
3. **올인원**: 날씨·물때·입질·기록·커뮤니티·예약을 한 앱에서 — 3-4개 앱 대체
4. **현대적 UX**: 2026년 수준의 디자인 — 한국 낚시 앱 중 유일하게 앱스토어 퀄리티

## 5. Job Stories (JTBD)

1. **When** 주말 출조 전날, **I want to** 내일 목표 어종의 입질 확률과 날씨를 한 화면에서 보고, **so I can** 출조 여부와 포인트를 빠르게 결정할 수 있다.
2. **When** 현장에서 물고기를 잡았을 때, **I want to** 사진 찍으면 어종·크기가 자동 기록되고, **so I can** 손맛에 집중하면서도 정확한 조과 기록을 남길 수 있다.
3. **When** 시즌이 끝나고 돌아보면, **I want to** 내 낚시 패턴(잘 잡히는 물때, 어종, 포인트)을 데이터로 보고, **so I can** 다음 시즌 전략을 세울 수 있다.
4. **When** 해상에서 주변 선박이 지나갈 때, **I want to** 대형 선박의 위치와 접근 여부를 실시간으로 확인하고, **so I can** 안전하게 낚시할 수 있다.
5. **When** 낚시 초보로 뭘 준비해야 할지 모를 때, **I want to** AI에게 장비·채비·포인트를 질문하고, **so I can** 검색 없이 바로 답을 얻어 준비할 수 있다.

## 7. 유저 플로우

### 시나리오 1: 출조 전 입질 확인
```
[앱 실행] → [홈: 오늘의 입질 예보 카드] → [탭: 어종별 입질 확률 확인]
→ [상세: 시간대별 피크 타임 그래프] → [결정: 출조 확정 or 연기]
```
- 홈 진입 2초 이내, 스크롤 없이 핵심 정보 노출
- 어종 선택 시 물때·날씨·입질 통합 뷰

### 시나리오 2: 현장 조과 기록
```
[하단 FAB(+) 탭] → [카메라 촬영] → [AI 어종 자동 식별]
→ [크기·무게 입력 (또는 음성)] → [날씨·위치·물때 자동 캡처] → [저장]
```
- 3탭 이내 기록 완료 목표
- 오프라인 시 로컬 저장 → 온라인 복귀 시 자동 동기화

### 시나리오 3: 시즌 분석
```
[하단 네비: 통계] → [월별/어종별 차트] → [베스트 조건 분석]
→ [공유 or 다음 시즌 계획]
```

## 8. 디자인 방향 (DESIGN.md 업데이트 필수)

- **디자인 도구 우선순위**:
  1. Stitch MCP — 새 디자인 시스템 생성
  2. godly.website / dribbble 벤치마킹
- **모바일 우선**: 모바일 퍼스트 (max-width 448px 기준, Capacitor 래핑)
- **톤/무드**: 프리미엄 아웃도어 — 다크 기반 + 시안/블루 악센트, 깔끔한 카드 UI
- **기존 DESIGN.md**: 완전 새 디자인이므로 drift-guard 스냅샷 갱신 필수
- **핵심 변경**:
  - 글래스모피즘 유지하되 더 세련된 그라데이션 + 마이크로 인터랙션
  - 바텀시트 패턴 강화 (지도 위 오버레이)
  - 카드 기반 레이아웃 + 제스처 네비게이션
  - 스켈레톤 UI 전면 도입 (로딩 스피너 제거)
