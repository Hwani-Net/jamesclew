# 소상공인 대출 캘린더 PRD

> 작성일: 2026-04-05 | 버전: v1.0

## 1. 배경 및 문제 정의

### 왜 만드는가?
- **현재 문제점**: 소상공인 정책자금은 분기별/월별로 접수 시작일이 다르고, 선착순 마감되어 **접수일을 놓치면 수천만원의 저금리 대출 기회를 잃는다.** 실제로 2026년 1분기 일반경영안정자금은 접수 후 3일 만에 마감되었고, 직접대출 일부는 3월에 이미 마감되었다.
- **시장 기회**: 기존 앱들(정책자금알리미, 소상공인 대출안내 등)은 상품 정보 나열에 그치며, **캘린더 기반 일정 관리 + 개인 맞춤 필터링 + 알림** 기능을 제공하는 앱이 전무하다. 캐시노트가 일부 일정을 제공하지만 대출 전용이 아니다.

### 해결하려는 핵심 문제 (1문장)
> 소상공인이 본인 조건에 맞는 대출 상품의 신청일·마감일을 놓치지 않도록 캘린더에서 한눈에 관리하고 알림받을 수 있게 한다.

### 우리의 차별화 포인트
- **캘린더 중심 UX**: 유일하게 캘린더 뷰로 대출 일정을 시각화 (마감=빨간색, 진행중=파란색, 예정=회색)
- **개인 맞춤 필터**: 업종·업력·매출·신용·지역 입력 시 자격 충족 상품만 표시
- **알림 예약**: 접수 시작 D-3, D-1, 당일 알림 → 선착순 마감 방지

## 5. Job Stories (JTBD)

1. When **대출 접수 시즌이 다가올 때**, I want to **내 조건에 맞는 상품의 접수일을 캘린더에서 확인**, so I can **선착순 마감 전에 미리 준비하고 신청**
2. When **특정 날짜를 클릭할 때**, I want to **그날 신청 가능한 대출 상품을 카드로 비교**, so I can **금리·한도·조건을 빠르게 비교하고 최적 상품 선택**
3. When **관심 대출 상품을 발견할 때**, I want to **접수 시작일에 알림을 예약**, so I can **접수일을 놓치지 않고 당일 즉시 신청**

## 7. 유저 플로우

### 시나리오 1: 캘린더에서 대출 일정 확인
```
[앱 진입] → [캘린더 월간 뷰] → [색깔 점으로 일정 확인] → [날짜 클릭] → [카드 리스트 표시] → [카드 탭하여 상세 보기]
```
- 캘린더: 파란 점(접수중), 초록 점(접수예정), 빨간 점(마감)
- 날짜 클릭 시: 하단에 카드 리스트 슬라이드업
- 카드: 상품명, 한도, 금리, 상태 배지, 자격 매칭 여부

### 시나리오 2: 필터링으로 내 조건 맞는 상품 찾기
```
[필터 아이콘 탭] → [업종/매출/신용/지역 입력] → [적용] → [캘린더에 자격 충족 상품만 표시] → [카드에 "자격 충족" 배지]
```

### 시나리오 3: 알림 예약
```
[카드 상세 보기] → [알림 예약 버튼] → [D-3/D-1/당일 선택] → [확인] → [브라우저 Notification 또는 이메일]
```
