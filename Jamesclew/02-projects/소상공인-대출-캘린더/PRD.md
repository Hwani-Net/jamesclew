# 소상공인 대출 캘린더 PRD

> 작성일: 2026-04-05 | 버전: v1.0

## 1. 배경 및 문제 정의

### 왜 만드는가?
- 소상공인 정책자금은 연 5.4조 원 규모이나, 자금별 신청일이 흩어져 있어 접수 시기를 놓치는 소상공인이 매우 많음
- 매월 접수일마다 소진공 홈페이지가 먹통되는 현상이 반복 (2024~2026년 매년 발생)
- 한 번 놓치면 다음 분기(3개월)까지 대기해야 하는 구조 → 재료비/임대료 등 자금 압박 심화
- 현재 캘린더 형태로 대출 일정을 관리해주는 전문 앱이 없음

### 해결하려는 핵심 문제 (1문장)
> 소상공인이 자신에게 맞는 정책자금 신청일을 놓치지 않고, 사전에 준비할 수 있도록 캘린더 기반 일정 관리와 알림을 제공한다.

## 2. 사용자 페인포인트 심층 분석

> 조사 출처: 연합뉴스(2024.01.29), 조선비즈(2026.03.16), 소통24 민원, 소진공 국민청원

| 페인포인트 | 빈도 | 현재 대안 | 대안의 한계 |
|-----------|------|----------|-----------|
| 자금별 접수일이 흩어져 있어 놓침 | 높음 | 소진공 홈페이지 공지, 유튜브 | 공지가 분기마다 별도 발표되어 한눈에 볼 수 없음 |
| 신청일에 서버 마비로 접수 실패 | 높음 | 새벽부터 대기, 반복 시도 | 수만 명 동시 접속, 10시 시작 30분 내 자금 소진 |
| 내 조건에 맞는 자금을 모름 | 중간 | 1357 전화상담, 유튜브 | 상담 연결 어렵고, 유튜브는 일반론 위주 |
| 한번 놓치면 3개월 대기 | 높음 | 없음 | 자금 압박 → 고금리 사채 전환 위험 |
| 복잡한 자격 조건 판단 어려움 | 중간 | 정책자금알리미 앱 | 자격 필터링 없이 정보 나열만 함 |

### 우리의 차별화 포인트
- **캘린더 뷰**: 모든 대출 상품 일정을 월별 캘린더에 시각화 (경쟁사 전무)
- **상태 표시**: 마감(빨간색)/진행중(초록색)/예정(파란색) 실시간 반영
- **개인화 필터**: 업종, 업력, 매출, 신용점수 기반 → 내게 맞는 상품만 표시
- **알림 예약**: 신청일 D-7, D-1, 당일 10분 전 푸시 알림

## 4. 타겟 사용자

| 항목 | 내용 |
|------|------|
| Primary 사용자 | 1인 소상공인 (카센터, 식당, 미용실 등 서비스업 종사자) |
| 사용 상황 | 정책자금 신청 시기가 다가올 때, 자금이 필요할 때 |
| 기존 대안 | 소진공 홈페이지 공지 + 유튜브 영상 + 지인 정보 |
| 전환 동기 | "접수일 놓쳐서 3개월 기다린 경험" → 알림만 있어도 해결 |

### 대상자 프로필 (종국이 동생)
- 업종: 카센터 (자동차 정비업)
- 개업일: 2025년 6월 (업력 ~10개월)
- 연 매출: 7,000만원 / 직원 0명
- 신용점수: 800점 미만 (NCB 기준)
- 소재지: 충북 청주시 상당구 (비수도권 우대 60% 대상)
- 기존 대출: 자동차 담보대출 약 3,000만원

## 5. Job Stories (JTBD)

1. **When** 다음 달 정책자금 접수일이 언제인지 확인하고 싶을 때, **I want to** 캘린더에서 한눈에 일정을 보고 싶다, **so I can** 미리 서류를 준비하고 접수 시간에 맞춰 대기할 수 있다
2. **When** 내 조건(업종/업력/신용점수)에 맞는 대출 상품이 뭔지 모를 때, **I want to** 내 정보를 입력하면 자격이 되는 상품만 필터링해서 보고 싶다, **so I can** 시간 낭비 없이 신청 가능한 것만 집중할 수 있다
3. **When** 접수일을 깜빡 잊을까봐 불안할 때, **I want to** 원하는 상품에 알림을 설정하고 싶다, **so I can** D-7, D-1, 당일에 미리 알림을 받아 놓치지 않을 수 있다

## 7. 유저 플로우

### 시나리오 1: 캘린더에서 대출 일정 확인
```
[앱 진입] → [캘린더 월뷰 표시: 날짜에 색상 도트]
  → [날짜 클릭] → [해당일 대출 상품 카드 리스트]
  → [카드 클릭] → [상품 상세: 조건/금리/한도/신청방법]
```
- 마감 상품: 빨간색 도트 + 카드에 "마감" 배지
- 진행중: 초록색 도트
- 예정: 파란색 도트

### 시나리오 2: 내 조건으로 필터링
```
[캘린더 상단 필터 버튼] → [필터 시트: 업종/업력/매출/신용점수 입력]
  → [적용] → [내 조건에 맞는 상품만 캘린더에 표시]
  → [필터 저장 (프로필로)]
```

### 시나리오 3: 알림 예약
```
[상품 카드에서 🔔 알림 버튼] → [알림 시점 선택: D-7/D-1/당일]
  → [설정 완료 토스트] → [예약된 알림 목록 확인 가능]
  → [알림 도착 시: 상품명 + 접수시간 + 바로가기]
```
