# /저장 — 세션 저장 + compact

현재 세션의 작업 내용을 저장하고 컨텍스트를 압축합니다.

## 실행 절차 (순서대로 모두 실행)

### 1. 옵시디언 세션 요약 저장
아래 경로에 세션 요약 마크다운 파일을 생성하세요:
`C:/Users/AIcreator/Obsidian-Vault/01-jamesclaw/harness/session-{날짜}-{주제}.md`

포함 내용:
- 세션 개요 (기간, 커밋 수, 주요 작업)
- 핵심 성과 (변경/추가/수정 목록)
- PITFALLS 신규 항목
- 다음 세션 작업 목록

### 2. 하네스 설계 문서 업데이트
하네스 구조가 변경되었으면:
`C:/Users/AIcreator/Obsidian-Vault/01-jamesclaw/harness/harness_design.md`
변경 이력 테이블에 날짜, 변경 내용, 근거 기록.

### 3. git commit + push
미커밋 변경사항이 있으면 커밋. push는 대표님 확인 후.

### 4. compact 자동 실행
위 1-3 완료 후 desktop-control로 `/compact`를 자동 입력합니다:

```
Step 1: 우측 Alt로 IME 영문 전환
  mcp__desktop-control__computer(action: "key", text: "alt_r")

Step 2: /compact 타이핑 + Enter
  mcp__desktop-control__computer(action: "type", text: "/compact")
  mcp__desktop-control__computer(action: "key", text: "Return")
```

⚠️ desktop-control MCP가 연결되어 있어야 합니다. 연결 안 되어 있으면 대표님에게 수동 입력 요청.
