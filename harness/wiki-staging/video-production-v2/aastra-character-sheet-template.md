# 캐릭터 시트 템플릿 — gpt-image-2 Master Sheet (10블록)

영상의 핵심: **모든 캐릭터 디테일을 한 장의 시트에 고정**. 이 시트가 이후 모든 컷의 기준점.

## 산업 표준 출처 체인

| 요소 | 원천 | 인용 |
|------|------|------|
| Character Model Sheet 표준 | **Disney Character Model Department (1937~1941)** [D23 공식](https://d23.com/walt-disney-and-the-creation-of-the-character-model-department/) | "1937년 Disney가 character consistency 위해 전담 부서 신설. 현재까지 적용되는 6요소(Turnaround/Expression/Pose/Props/Color/Annotations) 표준 정립" |
| 9블록 산업 적용 | [CGWire (현재 표준)](https://blog.cg-wire.com/character-sheet-animation/) + [SCAD 교육 자료](https://www.scad.edu/sites/default/files/PDF/Animation-design-challenge-character-sheets.pdf) | "Character sheet 6요소 + 학교/스튜디오별 확장" |
| 10번째 블록 (SIZE) | Disney 1937~ Character Size Chart 표준 | "scale reference vs other characters for proportion lock" |
| AI 시대 확장 (AI 아스트라 영상) | [YouTube 7oN5U0XFhNU](https://www.youtube.com/watch?v=7oN5U0XFhNU) | 한국어 시트 9블록 매거진 레이아웃 + HEX 락 |

> 우리 10블록 시트 = Disney 6요소 + AI 시대 확장 4블록(MOOD BOARD, ROLE CONCEPT, PARTS BREAKDOWN, **SIZE CHART**)

## 시트 1장 10블록 표준 레이아웃 (v2 보강)

| 블록 | 영문 라벨 | 내용 |
|------|-----------|------|
| **IDENT** | Identity | 이름 / 나이 / 국적 / MBTI / 역할 (프로필 박스 1개) |
| **FACE** | Expressions | 표정 6종 헤드샷 (기본·미소·놀람·경이·결의·감동) |
| **FIT** | Outfits | 메인 의상 / 디테일 2~3종 / 액세서리 (의상 4컷) |
| **BODY** | Full Body | 정면·측면·후면 전신 3종 |
| **POSE** | Poses | 동작 5종 (걷기/달리기/놀람/응시/웃음 등) |
| **PARTS** | Styling Breakdown | 의상 구성 분해 7종 (저고리·치마·꽃신·버선·노리개·댕기·머리장식) |
| **PROPS** | Story Props | 스토리 소품 3종 (부채·연등·복숭아 등) |
| **COLOR** | Color & Tone | HEX 4색 + 톤 콘셉트 1줄 |
| **MOOD** | Mood Board | 배경 무드 4컷 (장소·시간대·계절·분위기) |
| **SIZE** | Size Chart (NEW v2) | 다른 캐릭터 대비 신장 비율 차트 (Disney 1937 표준) |

> **시트키 2가지** — 이 두 가지가 일관성 99%를 결정:
> 1. **HEX 락** — 모든 색상 6자리 hex (#5B9FD5). "파란색", "sky blue" 같은 어휘 금지.
> 2. **미시 디테일** — "롱헤어" 금지 → "체스트 길이 웨이브 + 코랄 레드 리본". 길이/소재/위치 명시.

## 예시 1 — 서하린 (주인공 시트)

### IDENT
| 항목 | 값 |
|------|-----|
| 이름 (Name) | 서하린 (Seo Harin) |
| 생년월일 | 2004년 3월 15일 |
| 나이 | 22세 |
| 신장 | 164cm |
| 국적 | 대한민국 |
| MBTI | ENFP |
| 역할 | 한복 입고 경복궁 방문한 호기심 많은 여대생 |

### 프로필 (1문단)
> 밝고 호기심 많은 성격의 대학생. 전통 문화에 관심이 많아 한복을 입고 경복궁을 방문했다가 신비한 해치를 만나게 된다. 차분하면서도 모험을 즐기는 이중적인 매력을 가진 캐릭터. 표현력 있는 눈과 부드러운 웨이브 헤어가 특징.

### FACE — 표정 6종
1. 기본 (Neutral) / 2. 미소 (Soft Smile) / 3. 놀람 (Surprised) / 4. 경이 (Wonder) / 5. 결의 (Determined) / 6. 감동 (Moved)

### FIT — 의상 4컷
1. **메인 한복 (Main Hanbok, 메인)** — 크림 화이트 저고리 + 스카이 블루 치마 + 코랄 리본
2. 저고리 디테일 (Jeogori Detail) — 소매 꽃 자수
3. 치마 디테일 (Chima Detail) — 노리개 매듭
4. 액세서리 (Accessories) — 머리 리본, 노리개

### BODY — 전신 3종
정면 (Front) / 측면 (Side) / 후면 (Back)

### POSE — 동작 5종
걷기 / 달리기 / 놀람 / 응시 / 웃음

### PARTS — 스타일링 분해 7종
저고리 (Jeogori) · 치마 (Chima) · 꽃신 (Flower Shoes) · 버선 (Beoseon) · 노리개 (Norigae) · 댕기 (Deenggi) · 머리 액세서리 (Hair Accessories)

### PROPS — 스토리 소품 3종
부채 (Traditional Fan) · 연등 (Paper Lantern) · 복숭아 (Magical Peach)

### COLOR (4색 HEX 락)
- 하늘색 #5B9FD5 (sky blue)
- 코랄 레드 #E85A5A (coral red)
- 크림 화이트 #F8F0E0 (cream white)
- 버터엘로 #F2D86C (butter yellow)

> **톤 콘셉트**: 청량함, 산뜻함, 전통과 현대의 조화

### ROLE CONCEPT (역할 컨셉)
경복궁을 방문했다가 판타지 세계로 빠져드는 호기심 많고 감수성 풍부한 현대 여대생. 한국 전통의 아름다움과 현대 청춘의 싱그러움이 공존하는 캐릭터.

**추천 역할**:
- 판타지 어드벤처 히로인
- 감성 뮤직비디오 주인공
- 전통 × 현대 크로스오버 모델
- K-컬처 홍보 캐릭터

### MOOD BOARD
- 경복궁 정문 (낮 / 봄)
- 벚꽃 가지
- 푸른 하늘 + 흰 구름
- 노을 진 한옥 지붕

## 예시 2 — 해치 (보조 시트, 변신형 캐릭터)

### IDENT
| 항목 | 값 |
|------|-----|
| 이름 (Name) | 해치 (Haechi) |
| 별명 (Alt.) | 해태 (Haetae) |
| 종족 (Type) | 한국 신화 수호수 (Korean Mythological Guardian) |
| 출처 (Origin) | 고대 한국의 신수, 정의와 화재의 상징 |
| 크기 (Size) | 종합형 신장 80cm (어깨 위 높이) |
| 나이 (Age) | 천년+ (~1000 years old) |

### 프로필
> 천 년간 경복궁을 지켜온 신수. 평소엔 석상이지만 위험이 다가올 때 눈에서 빛이 새어 나오며 깨어난다. 정의를 지키고 거짓을 가려내는 신비로운 능력. 강한 외모지만 친근하고 현대 감성도 갖춘 사랑스러운 신수. 주인공을 판타지 세계로 인도하는 가이드.

### FORM STATES — 형태 변화 4단계
1. 석상 (Stone Statue) — 잠든 형태
2. 각성 중 (Awakening) — 균열 + 붉은 빛
3. **살아있는 해치 (Living Form, MAIN / 메인)**
4. 비행 중 (Flying)

### FACE — 표정 6종
1. 평온 (Calm) / 2. 장난 (Playful) / 3. 호기심 (Curious) / 4. 진지 (Serious) / 5. 웃음 (Laughing) / 6. 부드러움 (Gentle)

### BODY — 전신 3종
정면 (Front) / 측면 (Side) / 후면 (Back)

### POSE — 동작 5종
1. 달리기 (Running) / 2. 점프 (Jumping) / 3. 포효 (Roaring) / 4. 앉기 (Sitting) / 5. 장난 (Playing)

### DESIGN BREAKDOWN — 디자인 분해
- 갈기 (Mane) — 황금색 곱슬
- 비늘 (Scales) — 청록 무늬
- 눈 (Eyes) — 황금 동공
- 뿔 (Horns) — 작은 두 개
- 꼬리 (Tail) — 사자형 황금 끝
- 발 (Feet) — 4족 발톱

### ABILITIES — 능력 6종
- 변신 (Shape-shifting)
- 포털 생성 (Portal Creation)
- 비행 (Flight)
- 에지 (Edge / 영적 감지)
- 수호 (Protection)
- 통찰 (Foresight)

### COLOR (4색 HEX 락)
- 청록 #4A9B9B (turquoise)
- 황금 #E8C766 (gold)
- 코랄 레드 #E85A5A (coral red)
- 크림 #F8F0E0 (cream)

> **톤 콘셉트**: 신비로움, 친근함, 고귀함, 장난기

### SCALE REFERENCE
서하린 옆 1:0.6 비율. 어깨 위 80cm 높이의 사자만한 크기.

### MOOD BOARD
- 경복궁 광화문 앞 석상 위치
- 푸른 하늘 + 황금 햇살
- 한옥 지붕 끝에서 비행
- 꽃잎 흩날리는 광장

## 사용 절차

1. 본 템플릿을 복사하여 캐릭터별로 빈칸 채움
2. `aastra-master-prompts.md`의 CHARACTER PROMPT 섹션에 IDENT/FACE/FIT/COLOR 값을 삽입
3. GPT 5.5 image-2 (3:4 비율) 호출 → 마스터 시트 1장 생성
4. 결과물을 Read 도구로 시각 검증 → HEX 색상 일치 + 표정 6개 누락 없는지 확인
5. PASS 시 `characters/{name}/master_sheet.png`로 저장 → 이후 모든 컷의 reference로 사용
