### ⚠️ 침묵 의심 발견 — `self-evolve.sh` 경로 불일치

| 항목 | 내용 |
|------|------|
| **settings.json 참조** | `bash $HOME/.claude/hooks/self-evolve.sh --apply 2>/dev/null` (PreCompact) |
| **실제 소스 위치** | `harness/scripts/self-evolve.sh` |
| **배포 경로** | `$HOME/.claude/scripts/self-evolve.sh` (deploy.sh: scripts/* → ~/.claude/scripts/) |
| **hooks/ 존재 여부** | ❌ harness/hooks/self-evolve.sh 없음 (ls 직접 확인) |
| **silent fail 메커니즘** | `2>/dev/null` → bash: file not found 에러 무음 억제, exit 127 발생 |
| **영향** | 매 PreCompact 발화 시 self-evolve --apply 완전 침묵 실패 → 자기진화 누적 미적용 |
| **이전 감사 감지 여부** | ❌ 미감지 (파일 존재 = scripts/self-evolve.sh ✅만 확인, 경로 불일치 미검증) |

**비교 — 정상 패턴:**
```
# settings.json
"bash $HOME/.claude/scripts/audit-session.sh --compact"  ← scripts/ 참조

# harness/scripts/audit-session.sh 존재 → 배포: ~/.claude/scripts/audit-session.sh ✅
```

**문제 패턴:**
```
# settings.json
"bash $HOME/.claude/hooks/self-evolve.sh --apply 2>/dev/null"  ← hooks/ 참조

# harness/scripts/self-evolve.sh 존재 → 배포: ~/.claude/scripts/self-evolve.sh
# ~/.claude/hooks/self-evolve.sh = 존재 않음 → exit 127 → 2>/dev/null 무음 ❌
```

**제안 수정 (settings.json PreCompact):**
```json
"command": "bash -c 'bash $HOME/.claude/scripts/self-evolve.sh --apply 2>/dev/null'"
```
