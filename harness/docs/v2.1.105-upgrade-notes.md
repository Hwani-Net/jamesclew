# Claude Code v2.1.105 Upgrade Notes (2026-04-14)

## 반영된 변경사항

### PreCompact Hook Blocking
- `pre-compact-snapshot.sh`에서 옵시디언 저장 실패 시 `exit 2` → compact 차단
- P-007 (저장 없이 compact 금지) 규칙을 hook 레벨에서 강제
- `OBSIDIAN_VAULT` 미설정 시에는 차단하지 않음 (다른 프로젝트 호환)

### EnterWorktree path 파라미터
- 기존 worktree로 재진입 가능: `EnterWorktree(path: "existing/worktree")`
- `quality.md` Parallel Agent Safety 섹션에 문서화

### WebFetch 개선
- `<style>`/`<script>` 자동 제거 — CSS 무거운 페이지에서 콘텐츠 예산 소진 방지

### /proactive = /loop 별칭
- `/proactive`로 `/loop` 호출 가능

## 보류된 변경사항

### Plugin Monitors
- 매니페스트 `monitors` 키 — 백그라운드 모니터 자동 활성화
- 생태계에 사용 예시 없음, 공식 문서 확인 후 적용 예정
