## 증상
대표님이 과거 세션에서 제안한 핵심 기능 ("후보자가 전직 의원/공무원이면 성과 표시")이
PRD.md 어디에도 기록되지 않은 채 개발이 진행됨.
코드/데이터/타입 전 레이어에 걸쳐 미반영.

## 원인
세션 간 메모리 단절: 대표님 제안이 당시 대화로만 존재하고
PRD 또는 MEMORY.md에 저장되지 않음.
새 세션에서 PRD를 기준으로 파악하면 해당 기능 자체가 사라짐.

## 해결
발견 즉시 PRD에 반영하고 구현 진행.

## 재발 방지
대표님이 신규 기능을 제안하면 → 그 자리에서 PRD.md에 반영 + MEMORY.md에 project 메모리 저장.
"나중에 반영하겠다" 패턴 금지.
