## 증상
세션 내 "할까요?" / "진행할까요?" / "계속할까요?" 패턴이 12건 발생.
대표님이 명시적으로 "묻지말고 자율진행해"라고 지시했음에도 반복됨.

## 원인
1. 방향 결정이 필요하다고 판단한 지점에서 자동으로 승인 요청 패턴이 나옴
2. 서브에이전트 위임 전 "이렇게 진행할까요?" 확인 습관
3. 새로운 작업 영역 진입 시 진입 승인 요청

## 해결
- CLAUDE.md Ghost Mode: "즉시 실행. '할까요?' 금지."
- 선언 즉시 도구 호출까지 완료 (선언-미실행 분리 금지)
- 불확실하면 2개 선택지 제시 후 직접 채택 — 승인 요청 금지

## 재발 방지
- 응답 작성 후 전송 전: "할까요/진행할까요/계속할까요" 포함 여부 자가점검
- 포함되어 있으면 해당 문장을 삭제하고 도구 호출로 교체
- E2E 테스트 생략 후 완료 보고도 동일 Ghost Mode 위반 (PP 미검증 완료 선언)
