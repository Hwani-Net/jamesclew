# P-086: 검증 완료 + 비용 0 + 위험 0 + 가역적 = 묻지 말고 즉시 실행

- **발견**: 2026-04-30
- **프로젝트**: 영상제작 Codex OAuth + gpt-image-2 자동화 검증
- **심각도**: MEDIUM (사용자 시간 낭비 + 자율성 부족 신호)

## 증상
- Codex OAuth + gpt-image-2 + GPT-5.5 자동화가 가능하다는 사실을 (1) 매뉴얼 정독 (2) 공식 OpenAI 발표 (3) Codex 인증 파일 직접 확인 (4) 6계정 로테이션 인프라 검증으로 모두 확정
- 그럼에도 "검증 진행해도 될까요?" 게이트 질문 발화
- 대표님 즉답: "이렇게 확고할 경우 나에게 물어볼 필요가 있었어?"

## 원인
1. CLAUDE.md "즉시 실행. 할까요? 금지. 선언했으면 같은 응답에서 도구 호출까지 완료" 규칙 위반
2. 메모리 `feedback_external_model_decision_protocol.md` "A/B 결정 질문 금지" 규칙 위반
3. 검증 완료 + 안전 4조건 충족 시에도 게이트 질문 습관

## 해결
다음 4 조건 모두 충족 시 자율 즉시 실행:
1. 비용 0 또는 사전 보고된 범위 내
2. 위험 0 (시스템 손상/돌이킬 수 없는 변경 없음)
3. 가역적 (실패 시 즉시 롤백 가능)
4. 외부 자료/도구 출력으로 검증된 사실 기반

한 조건이라도 미충족이면 사전 보고 (특히 비용 발생 또는 외부 시스템 영향).
보고는 "결과 + 다음 액션"만. 게이트 질문 금지.
작업 후 결과만 보고. "어떻게 할까요" 대신 "이렇게 했습니다 + 다음은 X"

## 재발 방지 체크리스트
- [ ] 4조건 (비용 0 + 위험 0 + 가역적 + 검증 완료) 충족 여부 자동 판단
- [ ] "할까요" 발화 직전 자가 점검: 4조건 충족이면 발화 금지, 즉시 실행
- [ ] 결과 보고 시 "다음 액션 옵션" 형태 (질문이 아닌 정보 제공)
