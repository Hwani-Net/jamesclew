# P-166 — 쿠팡 파트너스 의류건조기 단독 무재고 (세탁기+세트만 노출)

## 증상
- MultiBlog 의류건조기-비교 페이지(2026-05-07) 5개 추천 제품의 쿠팡 파트너스 자동 매칭 결과 전부 부적합
- 정확 모델명 검색 (LG RC91U, 삼성 DV90B, 캐리어 클라윈드 9kg, LG RH8WV)도 NO_MATCH
- 매칭되는 제품은 모두 "세탁기+건조기 세트" (예: "LG전자 트롬 오브제 컬렉션 세탁기+건조기 세트 24kg") 또는 "캠핑용 저전력 드라이기 450W" 같은 다른 카테고리
- exclude "세탁기"/"세트"/"WF" 적용 시 NO_MATCH 100%

## 원인
- 쿠팡 마켓 현실: 의류건조기 단독 판매 제품이 거의 없음
- 주력 노출은 (a) 세탁기+건조기 세트 (b) 휴대용/캠핑용 저전력 드라이기 (c) 식기건조기 (의류 아님)
- 단순 모델명 매칭으로는 단독 건조기 검색 불가능
- 의류건조기-히트펌프 페이지(17/19/21kg 대용량)는 6/14 재검색에서 정확 매칭 확보 — 즉 대용량(17kg+)은 단독 판매됨

## 해결
- 의류건조기-비교 페이지(`dryer-heatpump-compare-2026-05-07`) **삭제** (2026-05-18 결정)
- 의류건조기-히트펌프 페이지(`dryer-heatpump-2026-05-07`)만 유지 (대용량 17/19/21kg + 위닉스 텀블 8kg 1건은 NO_MATCH로 항목 제거)
- 한국어 슬러그 폴더(drafts/2026-05-07-의류건조기-추천-비교-히트펌프/)는 보존 (역사·참조)

## 재발 방지
- 블로그 키워드 선정 단계에서 **쿠팡 단독 판매 여부 사전 검증** 필수
- 카테고리별 사전 검증 체크리스트:
  - 의류건조기: 단독 거의 없음 (세트만) → 대용량(17kg+)만 가능
  - 세탁기: 단독 OK
  - 식기세척기: 단독 OK
  - 에어프라이어: 단독 OK (단 일부 모델은 큐커 광고만 노출됨, NQ50T8539BS 등)
  - 전기밥솥: 단독 OK (단 일부 모델은 단종으로 무재고)
  - 모니터: 단독 OK (단 일부 모델은 단종)
  - 제습기: 단독 OK
  - 무선청소기: 단독 OK
  - 선풍기: 단독 OK
- 키워드 선정 시 카테고리별 쿠팡 검색 + 단독 vs 세트 비율 사전 점검
- `/blog-generate` Phase 0(키워드 검증)에 쿠팡 단독 판매 검증 단계 추가 제안
- P-164의 PARTNERS GATE를 보완해 카테고리 정합성도 검증하도록 v2 검토
